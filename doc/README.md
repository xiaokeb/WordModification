# Word 文本处理工具

## 📋 项目简介

一个功能完善的 Word 文档批量处理工具，支持批量替换 Word 文档中的文本，同时保持原文档的格式（字体、字号、颜色、段落样式等）完全不变。

## 📦 打包说明

**打包后 dist 目录包含**：
- ✅ `Word 文本处理工具.exe` - 可执行文件
- ✅ `README.md` - 项目说明

**分发方式**：将 dist 目录下的 exe 和 README.md 分发给用户
- 双击 exe 运行程序
- 打开 README.md 查看快速指南

## ✨ 核心功能

- ✅ **批量处理**: 支持批量读取 .doc 和 .docx 格式文件
- ✅ **智能替换**: 基于项目编号、公司名称、系统名称、人员名称等分类进行批量替换
- ✅ **格式保持**: 替换过程中保持原文档的所有格式不变
- ✅ **灵活选择**: 支持选择单个/多个文件或整个文件夹
- ✅ **进度显示**: 实时显示处理进度和当前文件
- ✅ **智能保存**: 按规则自动保存为 `原文件名 - 修改版.docx`
- ✅ **独立运行**: 打包为单个 exe 文件，无需 Python 环境

## 📖 使用说明

### 1. 选择文件
**三种方式选择文件**：

**方式一：浏览文件**
- 点击"浏览文件"按钮
- 选择一个或多个 Word 文件（.doc 或 .docx）
- 支持多选（按住 Ctrl 或 Shift）
- 路径显示：单个文件显示文件所在文件夹，多个文件若在同目录则显示该目录，否则显示文件数量

**方式二：浏览文件夹**
- 点击"浏览文件夹"按钮
- 选择包含 Word 文件的文件夹
- 自动识别文件夹内所有 .doc 和 .docx 文件
- 路径显示：显示所选文件夹的完整路径（最后一级文件夹）

**方式三：手动管理文件列表**
- 在"已选文件列表"区域查看已选文件
- **添加文件**：点击"添加文件 (+)"按钮，选择要添加的文件
- **删除文件**：选中文件后点击"删除文件 (-)"按钮，或双击文件直接删除
- 列表显示：仅显示文件名（不含路径），无列标题

**文件列表说明**：
- 显示已选择的所有文件名（不含路径）
- 支持滚动查看大量文件
- 双击文件可快速删除
- 自动过滤超过 50MB 的文件
- 路径信息在绿色标签后方显示

### 2. 设置替换规则
替换规则采用表格形式，包含三列：类别、原词、新词

**表格操作**：
- **编辑单元格**：双击"原词"或"新词"单元格即可编辑
- **保存编辑**：按 Enter 键或点击其他地方自动保存
- **取消编辑**：按 Escape 键取消
- **重置规则**：点击"重置为默认值"按钮恢复初始状态

**默认规则**：
- 项目编号：原词和新词均为"HT 合同编号 - 年份-XM001-01"
- 公司名称、系统名称、人员名称：默认为空

**注意事项**：
- 类别列（左侧）为灰色，不可编辑
- 原词和新词列（中间和右侧）为白色，可编辑
- 如不需替换某类别，保持其原词为空即可

### 3. 设置保存路径（可选）
- 点击"浏览"选择自定义保存路径
- 不选择则自动在源文件同级创建"修改版"文件夹

### 4. 开始处理
点击"开始处理"按钮，等待进度条完成即可。

## 📝 功能特性详解

### 格式保持技术

本工具采用先进的 run 级别操作技术：
- 分析段落中所有 run 的文本分布
- 精确定位待替换文本所在的 run
- 单 run 内替换：直接替换，保持格式
- 跨 run 替换：智能合并，复制格式属性
- 支持表格内的文本替换

### 支持的格式属性
- ✅ 字体（Font family）
- ✅ 字号（Font size）
- ✅ 粗体（Bold）
- ✅ 斜体（Italic）
- ✅ 下划线（Underline）
- ✅ 字体颜色（Font color）

## ⚠️ 注意事项

1. 本工具仅支持 .docx 格式（.doc 格式会提示不支持）
2. 建议先在小范围测试后再批量处理重要文档
3. 处理过程中请勿关闭程序
4. 确保文件未被其他程序占用

## 📊 性能指标

- 单个文档（100KB）：约 0.5 秒
- 批量文档（10 个，1MB 每个）：约 8 秒
- 内存占用：约 50MB
